# class-wp-upgrader-skins.php

## Overview

Deprecated shim that includes the upgrader skin classes. Use `class-wp-upgrader.php` and the specific skin files directly.

## Deprecation

Deprecated since WordPress 4.7.0. Use `class-wp-upgrader.php` instead.
