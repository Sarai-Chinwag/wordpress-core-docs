APIs for building WordPress admin interfaces.

**Source:** `wp-admin/includes/`

## Components

| Component | Description |
|-----------|-------------|

## Key APIs

- **Settings API** - Register settings pages, sections, and fields
- **Meta Boxes** - Add custom meta boxes to edit screens
- **List Tables** - Create sortable, filterable data tables
- **Admin Notices** - Display dismissible notices
