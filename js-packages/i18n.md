# @wordpress/i18n

Internationalization utilities for translating strings.

## Key APIs

### `__( text, domain? )`
Translate a string.

### `_x( text, context, domain? )`
Translate with context.

### `_n( single, plural, number, domain? )`
Translate plural forms based on `number`.

### `sprintf( format, ...args )`
Substitute placeholders in a translated string.

## Locale Data

### `setLocaleData( data, domain? )`
Registers translations for a domain.

### `getLocaleData( domain? )`
Returns loaded locale data.
